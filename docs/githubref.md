# GitHub Quick Reference
<br><br>

```
* git clone <repo-url> – Download a repository to your computer

* git pull – Get the latest changes from GitHub

* git branch – List all local branches

* git branch -a – List all local and remote branches

* git checkout -b <new-branch> – Create and switch to a new branch

* git checkout <branch-name> – Switch to an existing branch

* git switch <branch-name> – Newer, safer way to switch branches

* git switch -c <new-branch> – Newer, safer way to create a branch

* git status – Show changed files

* git add . – Stage all changes

* git add <file> – Stage a specific file

* git commit -m "message" – Commit staged changes

* git commit -am "message" – Stage and commit tracked files

* git push -u origin <branch-name> – Push a new branch to GitHub

* git push – Push new commits to an existing branch

* git merge <branch-name> – Merge another branch into your current one

* git rebase <branch-name> – Reapply commits on top of another branch (advanced)

* git fetch – Download updates without merging

* git pull --rebase – Pull updates and reapply your commits cleanly

* git stash – Temporarily save uncommitted changes

* git stash pop – Restore stashed changes

* git stash list – Show all stashes

* git restore <file> – Undo changes in a file

* git restore --staged <file> – Unstage a file

* git reset --soft HEAD~1 – Undo last commit but keep changes staged

* git reset --mixed HEAD~1 – Undo last commit and unstage changes

* git reset --hard – Reset your branch to the last commit (dangerous)

* git log – View commit history

* git log --oneline – Compact commit history

* git diff – Show line-by-line changes

* git diff --staged – Show staged changes

* git rm <file> – Remove a file and stage the deletion

* git mv <old> <new> – Rename or move a file

* git remote -v – Show connected remotes

* git remote add origin <url> – Connect a local repo to GitHub

* git tag – List tags

* git tag <name> – Create a lightweight tag

* git clean -fd – Remove untracked files and folders (dangerous)

```
